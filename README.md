# Utez Recursos App 📱

Aplicación móvil desarrollada en Kotlin con Jetpack Compose para la gestión de recursos y solicitudes.

## 🚀 Tecnologías
- Kotlin
- Jetpack Compose
- Navigation Compose
- MVVM

## 📂 Estructura
- ui → pantallas
- viewmodel → lógica
- data → repositorio y modelos
